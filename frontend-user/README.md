# MVD Frontend - User Version

Публичный интерфейс для граждан МВД России.

## Описание

Этот проект предоставляет публичный интерфейс для подачи обращений и проверки их статуса.

## Экраны

- `/complaint` - Подача обращения
- `/check-status` - Проверка статуса по номеру

## Запуск

```bash
flutter pub get
flutter run -d chrome --web-port=3000
```

Или используйте скрипт:
- `start.bat` (Windows)
